# RootedByte Portfolio

A modern IT specialist portfolio website with sections for web apps, troubleshooting tips, daily IT news, Faith + IT reflections, and contact.

## Publish for free with Vercel

1. Create a GitHub repository, for example `rootedbyte`.
2. Upload these files to the repository.
3. Go to Vercel and import the GitHub repo.
4. Keep defaults for a static site.
5. Deploy.

You will receive a free `your-project.vercel.app` URL.

## About .dev domains

A `.dev` domain is not normally provided free by hosting platforms. You buy it from a domain registrar or through a platform such as Vercel Domains. `.dev` requires HTTPS, and Vercel provides HTTPS when you connect the domain.

## What to edit before publishing

- Replace `RootedByte` with your preferred brand name.
- Replace `hello@example.com` with your real email.
- Replace GitHub and LinkedIn URLs.
- Add your real IT skills, certifications, projects, and screenshots.
- Add your first troubleshooting posts.
